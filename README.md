# Fotocélula
### Um projeto de eletrônica chamado de fotocélula, onde tem o intuito de ligar ou desligar uma carga com a presença ou ausência de luz, seja ela artificial ou natural.

* Esquemático
* Simulção
  * Protheus
  * SimulIDE
  * MultiSim
* PCB or Layout
* 3D
